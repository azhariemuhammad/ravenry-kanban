# revenry-kanban

This is an collaboration tool that organizes your projects into boards.

### Requirements:
  - React >= 16.6.0
  - NPM >= 8.x
  - yarn >= 1.6.0

### Run Dev
```sh
$ cd ravenry-kanban
$ yarn install
$ yarn start
```
### Run Test
```sh
$ cd ravenry-kanban
$ yarn install
$ yarn test
```

### Build
```sh
$ cd ravenry-kanban
$ yarn build
```
